Custom properties css exercise

  1. We have to use two dashes(hyphens) [--] or two underscore [ _ _ ] when defining a Custom Property's name

  2. Custom property names are case-sensitive. In other words, --cellColor and --cellcolor are considered two distinct property names. So we can't use spaces and slashes in the between of property sorthand in a Custom Property name.

  3. What possible values that we can assign to a new Custom Property are number, text, rem, percentage and pixel

  4. When is it useful to create Custom Properties using the `:root` or `html` selector? When is it not? It is useful when you want to apply it to another properties instead of repeating the values many times, so there is no need of using it if there is not any property that should use it.

  5. After we've defined a new Custom Property, we can recall or use that Custom Property by inserting the value of a custom property with var function.

  6. There are several ways to override or change a Custom Property. By using @media query, :root and html

  7. Custom Properties useful because it allows users to not repeat theirselves

  8. The only aspect (or part) of CSS syntax that we can use Custom Properties for is :root.
