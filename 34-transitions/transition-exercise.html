Transitions exercise

Transitions offer us the simplest form of animation in CSS.

The word transition means to 'change from one state to another'. In CSS, these transitions occur per element, per property.

  • The first state is the initial state, or how the element was to begin with.
  • The second state, or target state, is what the element will become.

Transitions take care of all the intermediate stages between those two states.

    There are four CSS properties that are included in transition shorthand property such us transition-property, transition-duration, transition-timing-function and transition-delay.

  2. What is the difference between a timing function, delay, and duration?

    The difference between them is that timing function takes care of how values are coming in time to be calculated for the css properties being affected by a transition effect, the delay is for the diration to wait befor starting a property's transition effect wheras duration of course controles how long a transition animation takes time to complete.

    They can be a problem when the user of the website that is animated very slow is in a rash. For example when you click a button, it may take 3 or 4 seconds to work depends on how long the transition animation takes time to complete(transition-duration).

    It is because we can not transform an element which has an auto property value to an other unit.

  9. How can we stagger transitions? i.e. make a number of child elements all start at different times?

    We ca achieve that by using different values of a same property with the parent element and the children. What this means is we could use the transition-delay and give it a different value in each siblings.
