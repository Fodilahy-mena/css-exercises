All the small things that we didn't cover about CSS

We haven't covered a number of topics that are really good to know, but each one of them probably didn't deserve their own exercise.

So, let's go through as many of the more important but small properties here as we can:

  1. A cursor Css property sets types of cursor when we over an element with mouse pointer. It gives types of cursor such us help sign, wait, crosshair, not-allowed, zoom-in, and grab depending on what you choose according to the meaning.

  2. A pointer-events CSS property sets diffent behaviours or events of a particular graphic element when the pointer is overing it.

  3. An outline CSS property takes care of the edge of an element which behaves like a border and has its longahand properties.

  4. A filter CSS property applies graphical effects like blur or color shift to an element. It is used to render images, backgrounds, and borderes.

  5. An appearance is useful when overriding something that sets by default.

  6. An object-fit CSS property takes care of how the content of a replaced element'scontents such us an image or video should be resized to fit in its container.

  7. A scroll-snap-align

  8. When you put a :focus pseudo class on an element, the element which is focused will be triggered when clicking on it, taps or select with keyboards tab key. It only applies to the focused element itself however we can use :focus-within if we want select an element which contains a focused element.

  9. A :visited pseudo class makes a diference between visited and unvisited element or a link depending on the value of the property that you give to it.
